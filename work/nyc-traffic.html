Cleaning noisy city data into usable analytics.

This project focused on processing and analyzing large-scale urban traffic + event data. The raw datasets were inconsistent, so the core work was cleaning, aligning, and structuring the data into analytical tables that can support downstream dashboards and modeling.

What I did

  • Standardized timestamps, locations, and identifiers across sources to make joins reliable.
  • Built a clean analytical layer (facts + dimensions) so questions can be answered without rewriting logic.
  • Validated edge cases (missing sensors, spikes, holiday effects) and documented assumptions.
