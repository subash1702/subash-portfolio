NYC Traffic Volume Analytics Pipeline

This project was about taking noisy city-scale data and turning it into tables that are actually usable. I focused on cleaning, aligning sources, and building analytical outputs that make trends obvious instead of confusing.
City-scale data, turned into clean analytical tables

What I did

  • Cleaned inconsistent raw datasets and aligned keys across sources
  • Built structured tables for analysis (not just one-off notebooks)
  • Designed outputs to support repeatable questions and comparisons

What I’d do next

If this went into production, I’d add scheduled loads, data tests, and a simple monitoring view: row counts, null checks, freshness, and a “did the distribution change?” signal.
