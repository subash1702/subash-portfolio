Nexora-NX

An AI-powered knowledge graph builder that turns messy text into a navigable intelligence map.
What it does

Nexora-NX converts unstructured text and PDFs into structured entity–relationship networks. It extracts entities and relationships as triplets, builds a graph, and lets you explore connections with interactive visualization and explainable answers.

Core capabilities
  • LLM-based triplet extraction with schema control and validation.
  • Graph construction and analytics using NetworkX (centrality, path discovery).
  • Interactive UI for exploring relationships and asking “how is X connected to Y?”
  • Evidence-based responses: answers come from graph paths and stored edge context.
Stack
  • LLM extraction + schema validation
  • NetworkX + graph algorithms
  • Streamlit + visualization layer
