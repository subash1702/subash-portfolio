Nexora-NX
Type AI Knowledge Graph Builder
Stack Python · LLM Extraction · NetworkX · Streamlit · PyVis
Focus Structured extraction + explainable graph reasoning

Turning messy text into a navigable intelligence map.

Nexora-NX converts unstructured text/PDFs into entity-relationship graphs. I built it to make information easier to explore: you can see what connects to what, run graph analytics, and ask questions like “How is X related to Y?” with evidence.

What it does

  • Extracts entities + relationships from text and produces clean triplets.
  • Builds a live NetworkX graph with metadata (evidence + confidence).
  • Runs ranking + path reasoning (PageRank, shortest paths).
  • Returns explainable answers based on the graph path, not vibes.
